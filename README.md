# **🛒 WhatBytes Marketplace \- Frontend Assignment**

A high-performance, pixel-perfect, and fully responsive E-commerce marketplace built with **Next.js 14** and **Tailwind CSS**. This project was developed as a technical assessment for the Frontend Developer role at WhatBytes.

## **🚀 Live Demo**

**View Project:** [https://frontend-whatbytes-assignment.vercel.app/](https://www.google.com/search?q=https://frontend-whatbytes-assignment-gules.vercel.app/)

## **✨ Key Features**

* **🎯 Dynamic Product Listing:** A clean grid layout displaying products with real-time updates.  
* **🔍 Smart Search:** Instant search functionality that filters products as you type.  
* **🏷️ Multi-Category Filtering:** Efficiently browse through Electronics, Clothing, and Home categories.  
* **💰 Price Range Filter:** Interactive range slider and manual input to filter products by budget.  
* **🛒 Robust Cart System:** \- Add/Remove items with a single click.  
  * Quantity control directly from the cart view.  
  * **Persistence:** Cart data is saved in localStorage, so your items stay even after a refresh.  
* **📱 Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens using Tailwind's mobile-first approach.  
* **✨ Featured Products:** Special layout highlighting flagship items (like the Smartphone card) with extended descriptions.  
* **📄 Product Detail View:** Dedicated view for each product with high-quality imagery and detailed specs.

## **🛠️ Tech Stack**

* **Framework:** [Next.js 14 (App Router)](https://nextjs.org/)  
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)  
* **Icons:** [Lucide React](https://lucide.dev/)  
* **State Management:** React Context API (for Global Cart State)  
* **Deployment:** [Vercel](https://vercel.com/)

## **⚙️ Setup & Installation**

Follow these steps to run the project locally:

1. **Clone the repository:**  
   git clone \[https://github.com/anujliwari/whatbytes-assignment.git\](https://github.com/anujliwari/whatbytes-assignment.git)

2. **Navigate to the directory:**  
   cd whatbytes-assignment

3. **Install dependencies:**  
   npm install

4. **Run the development server:**  
   npm run dev

   Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) in your browser.
